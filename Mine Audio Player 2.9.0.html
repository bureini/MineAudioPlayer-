Process your active track elements using the transcription node application:

  1. Click Open Lyric Transcriber to launch the target interface link context.
  2. Drop your track into the panel to calculate standard timestamp layouts.
  3. Copy the resulting synchronized blocks (e.g., [00:10.50] Lyric line...).
  4. Paste the copied text strings down into the text field block below.

About Mine Audio Player

How to Import Lyrics:

  1. Open the side menu drawer using the top-left menu button.
  2. Upload your audio tracks via the + Add Songs button first.
  3. Select the target song from the queue to make it the active track.
  4. Click + Import Lyrics to upload your local .me or plain .txt file.
  5. Or click + AI Lyrics to open the Lyric Transcriber framework interface.
